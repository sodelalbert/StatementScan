# Installation

Isntall dependencies and link the package to the python environment
```bash
pip install -e .
```

Development dependencies installation, installation with ZSH causes an error try doing
it from PyCharm or another terminal. 
```bash
pip install -e .[dev]
```

### Black manual check

```bash
black src
```


### MyPy Manual check

```bash
mypy src
```
